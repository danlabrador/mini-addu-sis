Grade Encoding Guidelines

  • Grades in the NEW Grading System must be encoded as A, B+, B, C+, C, D, F, FD, WP.
  • Non FYDP Grades in the OLD Grading System must be from 65 to 100.
  • FYDO Grades in the NEW Grading System must be encoded as A, B+, B, C+, C, D, F, FD, WP.
  • FYDP Grades in the OLD Grading System must be encoded as O - Outstanding, HS - Highly Satisfactory, MS - Moderately Satisfactory, F - Fair, P - Poor.
  • NE Grades will not be accepted. Students who have not taken exams must be provided the minimum grade. After the exam has been taken, the teacher should file a grade correction from the Registrar's Office.
